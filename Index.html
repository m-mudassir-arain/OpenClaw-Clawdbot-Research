OpenClaw (formerly Clawdbot/Moltbot) is an open-source AI agent platform designed to move AI from simple chat responses to real-world task execution. It connects AI models with tools, applications, files, and workflows to perform actions automatically.

1. What is OpenClaw?

OpenClaw is a self-hosted AI agent system. It connects:

🧠

AI Models

Claude, GPT, Gemini

🔧

Tools

Emails, Files, Browsing, APIs

🔗

APIs

Third-party integrations

💬

Messaging

Slack, WhatsApp, etc.

⚡

Automation

Workflow orchestration

Basic Flow:

User Request → AI Agent → Tools → Completed Action

Unlike normal chatbots, OpenClaw can perform tasks instead of only generating answers.

Windows Installation Guide

Using WSL2 — step by step

Step 1 Enable WSL2 on Windows ▼

Open PowerShell as Administrator and run:

wsl --install

Restart your computer after installation.

This installs:

  • Windows Subsystem for Linux
  • Ubuntu Linux
  • Required virtualization components
Step 2 Open Ubuntu ▼

Open Ubuntu from the Windows Start Menu.

Create:

  • Linux username
  • Linux password
Step 3 Update Linux Packages ▼

Run:

sudo apt update
sudo apt upgrade
Step 4 Install Required Development Tools ▼

Install required packages:

sudo apt install curl git build-essential

These tools are required for downloading and building projects.

Step 5 Install Node.js ▼

Install Node.js because many AI agent tools require a JavaScript runtime.

Check installation:

node -v
npm -v
Step 6 Download OpenClaw ▼

Clone the OpenClaw repository:

git clone <repository-url>

Move into the project:

cd openclaw

Install dependencies:

npm install
Step 7 Configure AI Model Access ▼

Configure required AI model API keys:

  • OpenAI API
  • Claude API
  • Gemini API

Store keys securely using environment variables.

Example:

export API_KEY=your_key_here
Step 8 Run OpenClaw ▼

Start the application using the project start command.

Verify:

  • Agent is running
  • Tools are connected
  • Skills are working correctly
